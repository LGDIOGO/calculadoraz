⚙️ Configure o Firebase

Para usar o app, você precisa criar um projeto Firebase gratuito e colar a configuração no arquivo index.html. Siga os passos:

  1. Acesse console.firebase.google.com
  2. Crie um projeto e ative Firestore e Authentication (e-mail/senha)
  3. Em Autenticação, adicione as contas dos 2 usuários
  4. Nas configurações do projeto, copie o firebaseConfig
  5. Cole no lugar indicado no início do <script> neste arquivo

Regras do Firestore (cole em Firestore → Regras):

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
